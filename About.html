About our website :

This is a website in which users can easily get programs of different programming language. In our website we have used 3 languages like C, C++ and JAVA. And we have embedded jdoodle online compiler for those three languages in which users can compile the programs by giving standard inputs and command line arguments. We have included login and signup page, it is not mandatory to login our website is open source. If user has to submit the programs into the website they have to register and then they can submit, so that when the other users visit our website they will get to know the contributors of the respective program. When the user submits code after login the code will be verified by the developers and then put it into the website with the contributor details. We have another page in which users can write their own programs and test their programs in jdoodle online compiler and they also can download the program into their system.
